# Pull Request Checklist

- [ ] Linked `TASK-YYYY-NNNN` is recorded.
- [ ] Risk tier matches `task.yaml`.
- [ ] Required approvals are complete and current.
- [ ] ADR impact is addressed and links are included if required.
- [ ] Evidence has been added or referenced.
- [ ] Rollback plan is current.
- [ ] Exceptions or waivers are linked if applicable.
- [ ] Review record has been updated.
