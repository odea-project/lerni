# Risk Assessment Template

## Risk Tier

Set one of: `low`, `medium`, `high`, `critical`.

## Rationale

Explain the classification against code impact, security, privacy, infrastructure, rollback, external effects, compliance, and agent autonomy.

## Affected Assets

- repository files or systems

## Hazards Or Failure Modes

- failure mode and impact

## Controls Required

- approval controls
- segregation controls
- evidence controls
- rollback controls

## Residual Risk

Describe the remaining risk after controls.

## Sign-Off

- assessor identity
- assessment timestamp
- approver confirmation if required
