# Tasks Directory

Each governed task lives in its own folder.

Active or historical tasks should use:

- `.aitask/tasks/TASK-YYYY-NNNN/`

Examples live under:

- `.aitask/tasks/examples/`

Each task folder must contain at minimum:

- `task.yaml`
- `spec.md`
- `risk.md`
- `execution-log.md`
- `review.md`
- `audit.md`
- `evidence-index.md`
- `closure.md`

The authoritative list of tracked tasks is in `.aitask/tasks/_index.yaml`.
